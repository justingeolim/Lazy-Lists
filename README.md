# Lazy Lists
Lazy Lists
Created a LazyColumn list of student names.

## Satisfies the following: 
Requirements
1.	Create a LazyColumn list of items.
2.	Create a screen that displays a list of 20 student names.
3.	The list must:
    o	Use LazyColumn
    o	Use the items() DSL
    o	Display each name inside a Card or Text
    o	Be scrollable

Sample Data
val students = listOf(
    "Alice","Bob","Charlie","Dana","Eric",
    "Fatima","Grace","Hiro","Isabel","Jack",
    "Karen","Luis","Maya","Nate","Olivia",
    "Priya","Quinn","Ravi","Sara","Tom"
)
